# Utils Scripts

- `plot_unitigs_with_minimap.py`: Maps unitigs to a reference genome and produces mapping/coverage outputs (including PAF).
- `mapped_unmapped_ids.py`: Builds mapped and unmapped unitig ID lists from unitig FASTA + mapping PAF.
- `run_blast_public_db_novelty.py`: Summarizes BLAST novelty classes and writes the human-readable Supplementary File 5 output.
